Represents a property whose value is available when HasValue is true.

Declaration

Property Value
Type Description
T
Remarks

You must await property HasValue before accessing this property.

Examples
 if (await maybe.HasValue)
 {
      // Safe to use.
     Console.WriteLine(maybe.Value)
 }
