Struct Result<T, TError>

A data-structure commonly used for error-handling where only one value can be present. Either it's TError or it's T. Which makes it possible to handle error without throwing exceptions. Inspired by 'Haskell's Either a b' and FSharps 'Result<T, TError>'.

Null values are not allowed and therefore excessively be checked before beginning an Result<T, TError> expression chain.
